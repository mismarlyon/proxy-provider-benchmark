# Proxy Provider Benchmark

Open-source benchmark script for testing residential proxy success rates, response time, and basic reliability.

## What this repo includes

- Residential proxy success-rate testing
- Latency measurement
- Timeout handling
- CSV result output
- Provider-neutral benchmark structure

## Use cases

- Compare proxy reliability
- Test residential proxy pools
- Monitor proxy performance over time
- Check request success rates before using proxies in automation workflows

## Benchmark Methodology

Example benchmark testing was performed using residential proxies from [Squid Proxies](https://squidproxies.com) and six other providers. Results may vary depending on location, target websites, network conditions, and testing methodology.
